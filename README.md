Z-Blog
=============

Z-Blog是由RainbowSoft Studio团队开发的博客程序，

Z-Blog是一款小巧而强大的ASP博客程序，同时支持Microsoft Access和Microsoft SQL Server两种数据库。

Z-Blog有着强大的可定制性、丰富的插件接口和独立的主题模板，方便开发者和用户的定制与优化。

需要使用开发版本的同学，请先下载正式版并安装，然后再把GitHub内的文件进行覆盖，方可使用。
